#  Weather Dashboard

A modern and responsive weather dashboard built using React and Material UI.  
This application allows users to search for any city and view real-time weather information with a clean glass-style interface.

---

##  Features

- Search weather by city name
- Real-time temperature data
- Min / Max temperature
- Humidity and “Feels Like” information
- Dynamic background based on weather condition
- Weather-based icon rendering
- Glassmorphism UI design
- Smooth transitions and responsive layout
- Secure API key using environment variables

---

##  Tech Stack

- React (Vite)
- Material UI
- OpenWeather API
- CSS + MUI `sx` styling
